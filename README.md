## 📌 Features

* ✅ Multiple Choice Questions (MCQs)
* ✅ Real-time score calculation
* ✅ Timer-based quiz (optional)
* ✅ Instant feedback (correct/incorrect)
* ✅ Result summary at the end
* ✅ Responsive design (mobile + desktop)
* ✅ Easy to add or modify questions

## 📝 How It Works

1. User starts the quiz
2. Questions are displayed one by one
3. User selects an answer
4. App checks correctness
5. Score updates in real-time
6. Final result is shown
